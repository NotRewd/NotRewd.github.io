Pixel Shifter Map Creator

In Pixel Shifter you are able to create your own maps using JSON files. These are the steps you need to take to create your first very own map. If you already know how to create the map, here's the full documentation.

File Creation

Create a file named "earlyaccesslevel.json" in your desktop and drag it over to the game folder "Pixel Shifter_Data\StreamingAssets\mapdata\"

File Editing

Now that you have your file created, we can begin editing it. Open it with your favourite editor and try to replicate this template. You can also add more objects by putting a comma after "}" and adding another {  } with parameters in it.

Play The Level

Now you can play the level and see that a white cube spawned after one second.
